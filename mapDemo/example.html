🗺️ 高德地图绘制工具

一个功能强大的地图绘制和坐标获取工具

🚀 快速开始

点击下面的按钮开始使用地图绘制工具：

打开地图绘制工具

✨ 主要功能

  • 支持绘制多边形、矩形、圆形
  • 实时显示绘制点的坐标
  • 获取图形角点的经纬度坐标
  • 导出绘制数据为JSON格式
  • 现代化的用户界面设计
  • 响应式设计，支持移动端

📝 使用步骤

  1. 获取API密钥：访问 高德开放平台 注册并获取JavaScript API密钥
  2. 配置密钥：在 config.js 文件中填入您的API密钥
  3. 开始使用：打开 index.html 文件即可开始绘制

🔧 配置示例

// config.js const MAP_CONFIG = { API_KEY: 'your_amap_api_key_here', MAP_OPTIONS: { zoom: 13, center: [116.397428, 39.90923], // 北京天安门 mapStyle: 'amap://styles/normal' } };

📊 导出数据格式

{ "timestamp": "2024-01-01T12:00:00.000Z", "shapes": [ { "type": "polygon", "coordinates": [ [116.123456, 39.123456], [116.234567, 39.234567], [116.345678, 39.345678] ] } ] }
